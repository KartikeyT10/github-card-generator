# LinkedIn Post Draft

I recently attended the GeeksforGeeks workshop **Building Personalized Agents With ADK, CP And Memory Bank**, and used it as a hands-on opportunity to build and deploy a complete AI-powered developer tool.

During the workshop, I worked on a **GitHub Dev Card Generator**: a web app that takes any public GitHub username, fetches useful profile and repository data, analyzes the developer profile, and generates a clean shareable developer card.

What I built:

- A FastAPI backend for the API layer
- GitHub profile and repository data fetching
- Gemini-powered profile analysis with a fallback path
- A simple frontend for generating, previewing, copying, and downloading cards
- Docker support for local development
- Vercel deployment for a working public link

Live project:
https://github-card-generator-one.vercel.app

GitHub repository:
https://github.com/KartikeyT10/github-card-generator

Workshop reference:
https://www.youtube.com/live/mIeMRlWoCf0

This workshop helped me understand how modern AI-assisted development can speed up the process from idea to working product, especially when combined with clear prompts, modular backend design, API orchestration, and deployment-first thinking.

The biggest takeaway for me was that AI tools are most powerful when they are used with engineering discipline: breaking the project into phases, verifying each layer, handling errors, securing environment variables, and deploying something that people can actually try.

Grateful to GeeksforGeeks and the workshop mentors for the practical session.
